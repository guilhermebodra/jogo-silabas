# 🎪 Aventura das Sílabas

Um app educativo e divertido para crianças de 6 anos aprenderem a ler e escrever através de um jogo interativo de ordenação de sílabas.

## 🎮 Como Funciona

1. **Veja o animal/objeto** - Uma imagem colorida aparece na tela
2. **Ouça a palavra** - Clique no botão para ouvir a palavra pronunciada
3. **Monte a palavra** - Clique nas sílabas na ordem correta para montar a palavra
4. **Verifique** - Clique em "Verificar" para checar se acertou
5. **Ganhe pontos** - Cada acerto ganha uma estrela! ⭐

## ✨ Características

- 🎵 **Música de fundo alegre** - Cria um ambiente divertido e motivador
- 🎉 **Sons de acerto** - Notas musicais alegres quando você acerta
- ❌ **Sons de erro** - Feedback sonoro quando erra
- 🎨 **Interface colorida** - Design pensado especialmente para crianças
- 📱 **Responsivo** - Funciona em tablets, celulares e computadores
- 🐄 **16 palavras diferentes** - Animais da fazenda e princesas
- 🧩 **Sílabas extras** - Aumenta a dificuldade do jogo

## 🚀 Como Usar

Basta abrir o arquivo `index.html` no navegador. Não precisa de instalação!

## 💡 Dicas para Pais e Educadores

- **Comece devagar** - Deixe a criança se familiarizar com o jogo
- **Pratique regularmente** - 10-15 minutos por dia é ideal
- **Celebrate wins** - Comemore cada acerto para motivar
- **Adicione palavras** - Você pode editar o arquivo para adicionar mais palavras

## 🛠️ Personalizando o Jogo

Para adicionar mais palavras, abra o arquivo `index.html` em um editor de texto e procure por `gameData`. Cada palavra segue este formato:

```javascript
{
    animal: '🐄',
    word: 'VACA',
    syllables: ['VA', 'CA'],
    extraSyllables: ['PA', 'MA', 'TA']
}
```
